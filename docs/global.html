TruthPath

If `circuit` is BoolVar, TruthPath is a Boolean. Otherwise, `circuit` is a AllQuantifier | AnyQuantifier, and TruthPath is an array of TruthPaths that corresponds to its children.
E.g., ALL( ANY(A,B), C ).
Suppose A is false, B is true and C is true. Then TruthPath is [[false, true], true]
Type:
  • Boolean | Array.<TruthPath>

TruthValue

Represents a truth value
- 'T': True
- 'F': False
- 'U': unknown (user may have clicked "don't know")
- null : user hasn't clicked on anything
Type:
  • null | 'U' | 'T' | 'F'
